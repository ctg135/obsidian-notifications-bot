# Telegram bot for sending notifications from Obsidian



## Instalation

### Configuration file

File `pass.json` with authorization data for WebDav server looks like this:

```json
{
    "vault_name": "name",
    "bot_id": "###",
    "user_id": "###",
    "server": "https://webdav.server.net",
    "login": "username",
    "password": "###"
}
```
